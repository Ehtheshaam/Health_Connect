# 🌍 HealthConnect – Smart Rural Healthcare Access App

## 📌 About the Project

HealthConnect is a multilingual healthcare application designed to connect rural communities with nearby hospitals and essential medical services.
It is built to function effectively in low-network conditions, ensuring accessibility for users in remote areas.



## 💡 Problem Statement

People in rural areas often face:

* Limited access to hospitals and healthcare services
* Language barriers in using digital applications
* Poor internet connectivity
* Delay in receiving medical assistance


## 🚀 Solution

HealthConnect provides a lightweight and user-friendly platform that:

* Works smoothly even on low internet connectivity
* Supports multiple regional languages
* Helps users find and connect with nearby hospitals
* Provides basic health-related assistance


## ⚙️ Key Features

* 🌐 Multilingual Support (English, Telugu, Tamil, Hindi, etc.)
* 📡 Optimized for Low Network Usage
* 🏥 Nearby Hospital Connectivity
* 📍 Location-Based Services
* 🤖 Basic Health Assistance
* 📱 Simple and Easy-to-Use Interface

## 🛠 Tech Stack

Frontend: React Native (Expo)
Language: TypeScript
State Management: Custom Hooks / Global Store
Internationalization (i18n): Multilingual support using JSON-based localization
Navigation: Expo Router
Package Manager: npm

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project in VS Code
3. Install dependencies

   ```
   npm install
   ```
4. Start the application

   ```
   npx expo start
   ```
